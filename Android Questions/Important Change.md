## Android 11

Permissions dialog
Bỏ quyền: WRITE_EXTERNAL_STORAGE, WRITE_MEDIA_STORAGE

- Android/data/
- Android/obb/

Truy cập file bằng: intent action ACTION_OPEN_DOCUMENT_TREE
- Ổ lưu trữ nội bộ
- Thẻ SD
- folder download
